# Time Tracker Pro

A Chrome extension (Manifest V3) that shows a floating time-tracking widget with
tasks & goals, focus mode, idle auto-stop, overrun alerts, history, summary
charts, and CSV / Google Sheets export.

> **Restructure in progress.** The codebase is being modularized in phases —
> see [RESTRUCTURE.md](./RESTRUCTURE.md) for the full plan and status.

## Project layout

```
src/
  core/      framework-free logic (unit-tested): time, model, summary, export,
             timer (state transitions), store (chrome.storage), log (shared
             day-log read-modify-write)
  shared/    constants (defaults, colors, storage keys) + message types
  ui/
    shared/  styles.css, markup.js, controller.js (mountTrackerUI) — the
             tracker UI, reused by the surfaces below
    popup/   the toolbar popup (primary UI) — popup.html + popup.js
    options/ settings page (tasks/goals, alerts, idle, Sheets, widget toggle)
    widget/  the floating widget (content script) — mounts the shared UI in a
             Shadow DOM and adds drag/focus/collapse; opt-in (see below)
  background/ service worker — owns the running timer, overrun alarms
              (chrome.alarms), idle auto-stop (chrome.idle), and the opt-in
              widget registration (chrome.scripting)
  icon.png   extension/notification icon (generated by scripts/gen-icon.mjs)
test/        node:test unit tests for core/
build.mjs    esbuild bundler -> dist/
```

## Develop

```bash
npm install        # install esbuild (dev dependency)
npm run build      # bundle src/ -> dist/
npm run watch      # rebuild on change
npm test           # run unit tests (no deps, uses node:test)
```

## Load in Chrome

1. `npm run build`
2. Open `chrome://extensions`, enable **Developer mode**
3. **Load unpacked** → select the generated **`dist/`** folder

Click the toolbar icon for the **popup** (the primary UI). Open **Settings**
(the gear, or the extension's options page) to manage tasks/goals and alerts.
The floating widget is **opt-in**: enable "Show the floating widget" in Settings
and grant access, and it will appear on pages you visit afterwards.
